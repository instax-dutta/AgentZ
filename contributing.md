# Contributing to AgentZ

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/instax-dutta/AgentZ.git
   ```

2. Navigate to the project directory:
   ```
   cd AgentZ
   ```

3. Create a new branch for your changes:
   ```
   git checkout -b new-feature
   ```

4. Make your changes and commit them:
   ```
   git add .
   git commit -m 'Your commit message'
   ```

5. Push your changes to the remote repository:
   ```
   git push origin new-feature
   ```

## Making a Pull Request

1. Go to the AgentZ repository on GitHub.
2. Click on the 'Pull requests' tab.
3. Click on the 'New pull request' button.
4. Select the branch you want to merge (e.g., `new-feature`).
5. Add a title and description for your pull request.
6. Click on 'Create pull request'.

## Codebase Guidelines

- Follow the existing coding style and conventions used in the project.
- Write clear and concise commit messages.
- Use meaningful names for branches and pull requests.
- Be patient and respectful in the pull request review process.
